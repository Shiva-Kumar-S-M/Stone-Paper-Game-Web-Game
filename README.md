Stone Paper Scissors – Web Game

## Overview

This project is a browser-based implementation of the classic Rock Paper Scissors game built using HTML, CSS, and JavaScript. It focuses on combining user interface design with interactive game logic.

The objective is to understand how frontend components and JavaScript logic work together to create a responsive user experience.

---

## Problem Statement

Building interactive applications requires handling user input, updating the interface dynamically, and implementing logical decision-making. This project demonstrates how these elements come together in a simple game.
